---
title: Installation
description: Install Minions Taxonomy packages
---

## TypeScript / Node.js

```bash
npm install @minions-taxonomy/sdk minions-sdk
```

## Python

```bash
pip install minions-taxonomy
```

## CLI

```bash
npm install -g @minions-taxonomy/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
