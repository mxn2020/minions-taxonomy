# @minions-taxonomy/cli

CLI for Minions Taxonomy — Organizational metadata — groups, categories, tags, namespaces, statuses, and priorities

## Installation

```bash
npm install -g @minions-taxonomy/cli
```

## Usage

```bash
taxonomy --help
```
